SpringのDebugメモ¶

本章ではSpring環境の構築およびアプリケーション開発時に詰まった内容を備忘録として残す。

Talend API Tester¶

httpメソッドでの通信を開放している場合に、httpsでリクエストを投げている可能性がある。

対処内容¶

Talend API Testerのプロトコルがhttpになっていることを確認する。 httpsになっていればhttpに変更する。

InteliJ¶

Mapper XMLファイルが読み込まれず、「Could not autowire.」とエラーを出力する¶

事象および原因¶

1
Could not autowire. No beans of 'AccountRepository' type found.

resources配下にMapper XMLファイルを作成して、MyBatisよりRepositoryを自動出力している場合に「Could not autowire.」とエラーを出力する。 ただ、アプリケーションを実行すると正常に動作する。

原因は、MyBatisを用いてRepositoryを作成する場合、IDEの静的解析では実体クラスが作成されておらずIDEが実体クラスを検知できないため。

対処内容¶

（2022/04/17 検証中） 対象のRepositoryインタフェースに@Repositoryアノテーションを付与することで、エラーを回避できる。
