=====================================================
SpringBoot
=====================================================
本章ではSpringBootのチュートリアル時の作業メモを残す。


前提事項
--------
* Windows11
* JDK 11
* Spring Boot 2.6.6

Spring Bootの概要
-----------------

プロジェクトの初期化
----------------------
Spring initializrに必要項目を入力して、generateボタンを押下すると、
zip形式のプロジェクトファイルがダウンロードできる。

https://start.spring.io/


Spring Bootのコマンド
-----------------------
Spring Bootアプリケーションを実行する。

.. sourcecode:: bash
   :linenos:

   mvn spring-boot:run
